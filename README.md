# How to run
- Setup .env with these keys:
```
BOT_TOKEN=
MODERATOR_ROLE_ID=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_API_KEY=
CALENDAR_ID=
DISCORD_NOTIFICATION_CHANNEL_ID=

```
- `pip install -r requirements.txt`
- `python3 main.py`
